# Object Types / Data types

- Number : 1234, 3.1215, 2+4j, 0b111, Decimal(),
Fraction()
- String : "Tushh", 'tushar', b'a\x01c', u'sp\xc4m'
- List : [1,[2,'Three'],4,5], list(range(10))
- Tuple : (1,'spam', 4, 'U'), tuple('spam'), namedtuple

- Boolean : True, False
